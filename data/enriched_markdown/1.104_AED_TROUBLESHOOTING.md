OBJECTIVE:
To identify problem conditions that may be encountered during use of the AED, and provide possible solutions.

| Observation | Possible Cause | Action |
|---|---|---|
| CONNECT ELECTRODES voice prompt is heard | Poor AED Electrode-to-skin contact | 1. Firmly press AED Electrodes on patient's skin.<br>2. If this does not correct problem and large amount of chest hair present Swiftly remove AED Electrodes and any attached chest hair.<br>3. Place new set of AED Electrodes on chest. |
| | AED Electrodes are dry or damaged | Replace the AED Electrode pads. |
| | AED Electrodes are not removed from the liner | Remove the AED Electrodes from the liner and apply them to the patient's chest. |
| CHECK CONNECTOR AND ELECTRODES voice prompt is heard | Connection to the AED is inadequate | Check to be sure that the AED Electrode connector is inserted completely. |
| MOTION DETECTED and STOP MOTION voice prompts are heard | Patient movement because of breathing | Check patient for normal breathing. |
| | CPR being performed during analysis | Stop CPR during analysis. |
| | Electrical/radio frequency interference | Move suspected devices away from the AED when possible. |
| STATUS DISPLAY is blank | The AED has been turned on | Normal condition when AED is in use |
| MAIN DISPLAY blank and ON LED lit | LCD not operating properly<br>MAIN DISPLAY not functioning properly | 1. AED may still operate. If needed, continue to use device to treat patient.<br>2. Notify MCC-H. |
