1.105 CHOKING
(MED CL/HTV5 - ALL/FIN 2/Paper on ISS/T) Page 1 of 1 page

OBJECTIVE:
To rapidly clear airway obstruction.

-> 1 CHECK PATIENT RESPONSE
Is patient awake?
-> YES -> 2
-> NO -> 3

2 PERFORM ABDOMINAL THRUSTS
1. Position behind patient.
2. Make fist.
3. Place fist above belly button but below ribcage. (Figure 1)
4. Pull sharply inward and upward.
5. Repeat until object clears or patient becomes unconscious. 
-> PATIENT BECOMES UNCONSCIOUS -> 3
-> OBJECT CLEARS -> 4

3 CLEAR AIRWAY/GIVE BREATHS
1. Open airway using Head tilt-Chin lift. (Figure 2)
2. Inspect for foreign object and remove, if easily accessible.
3. Pinch nose and attempt to give breaths.
Go to 1.102 AED ASSISTED CPR, all (SODF: MED CL: EMERGENCY).

-> 5
