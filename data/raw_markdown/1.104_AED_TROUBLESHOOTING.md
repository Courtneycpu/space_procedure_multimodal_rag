OBJECTIVE:
To identify problem conditions that may be encountered during use of the AED, and
provide possible solutions.

| Observation | Possible Cause | Action |
| --- | --- | --- |
| CONNECT ELECTRODES<br>voice prompt is heard | Poor AED Electrode-to-skin<br>contact | 1. Firmly press AED<br>Electrodes on patient’s skin.<br>2. If this does not correct<br>problem and large amount<br>of chest hair present<br>Swiftly remove AED<br>Electrodes and any<br>attached chest hair.<br>3. Place new set of AED<br>Electrodes on chest. |
|  | AED Electrodes are dry or<br>damaged | Replace the AED Electrode<br>pads. |
|  | AED Electrodes are not<br>removed from the liner | Remove the AED Electrodes<br>from the liner and apply them to<br>the patient’s chest. |
| CHECK CONNECTOR AND<br>ELECTRODES voice prompt<br>is heard | Connection to the AED is<br>inadequate | Check to be sure that the AED<br>Electrode connector is inserted<br>completely. |
| MOTION DETECTED and<br>STOP MOTION voice prompts<br>are heard | Patient movement because of<br>breathing | Check patient for normal<br>breathing. |
|  | CPR being performed during<br>analysis | Stop CPR during analysis. |
|  | Electrical/radio frequency<br>interference | Move suspected devices away<br>from the AED when possible. |
| STATUS DISPLAY is blank | The AED has been turned on | Normal condition when AED is<br>in use |
|  | LCD not operating properly | Notify MCC-H. |
| MAIN DISPLAY blank and ON<br>LED lit | MAIN DISPLAY not functioning<br>properly | 1. AED may still operate. If<br>needed, continue to use<br>device to treat patient.<br>2. Notify MCC-H. |
