Question    : The AED just told us to start CPR, but I think I might feel a pulse. What is the check and when do we stop compressions?

Answer:
After the AED prompts 'Start CPR', perform a 10-second pulse check. 

- If a pulse is detected: Stop chest compressions and assist breathing as necessary.
- If no pulse is detected: Continue CPR as directed by the AED.

Answer:
1. Perform a 10-second pulse check after the AED prompts to start CPR.  
2. If a pulse is detected:  
   - Stop chest compressions.  
   - Assist breathing as necessary.  
   - If a second rescuer is assisting, stop medication administration.  
   - Proceed to block 13, page 5, Respiratory Assessment.  
3. If no pulse is detected:  
   - Continue performing CPR (30 chest compressions for every 2 breaths, 1 second each breath).  
   - Repeat the pulse check after each "Start CPR" prompt from the AED.
